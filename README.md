## 🚀 Projects Overview

### Project 1 — Rule-Based AI Chatbot
> **Track:** Control Flow & Logic | **File:** `.py`

A deterministic chatbot built with pure Python. Uses dictionary-based O(1) lookup instead of if-elif ladders, runs in a continuous `while` loop, sanitizes input, and exits gracefully.

**Key Skills:** Control flow, dictionary lookups, input sanitization, IPO model  
📁 [View Project →](./Project1_Chatbot/)

---

### Project 2 — Data Classification Using AI
> **Track:** Supervised Learning | **File:** `.ipynb`

A full ML pipeline that classifies Iris flowers using K-Nearest Neighbors. Includes feature scaling, elbow method for optimal K selection, and professional evaluation via Confusion Matrix and F1 Score.

**Key Skills:** KNN, StandardScaler, train/test split, F1 Score, Confusion Matrix  
📁 [View Project →](./Project2_Data_Classification/)
